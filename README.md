# 数当てCLIゲーム（Python）

## 概要
1〜20の整数を当てるシンプルなコマンドラインゲームです。入力に対して「高すぎ」「低すぎ」「正解！」のフィードバックを返します。

## 必要環境
- Python 3.9 以上 推奨

## 実行方法
```bash
python3 -m src.main
```

## ルール
- ゲーム開始時に1〜20の乱数が生成されます。
- 整数を入力してください。
- 正解するまで繰り返します。範囲外/非整数は再入力を促されます。

## ファイル構成
- `src/logic.py`: 判定ロジック（テストしやすい純粋関数）
- `src/main.py`: CLIエントリポイント

## Web版（HTML/JavaScript）

### 実行方法（ローカル）
1. `web/index.html` をブラウザで開くだけで動作します。
2. もしくは簡易サーバを利用:
   - Python: `python3 -m http.server` を実行して `http://localhost:8000/web/` を開く
   - Node(npx): `npx serve web` など

### 構成
- `web/index.html`: 画面・入力UI
- `web/styles.css`: スタイル
- `web/script.js`: ゲームロジック（入力検証/フィードバック/リセット）
